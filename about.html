Project Guidelines
The Website is structured as a database with different spaces I used in my observation mapped across. This Visual Essay is a part of curriculum for Visual History at The New School. The objective was to pick a topic from the things discussed during class and make a narrative using research, photographs and observation. I picked this topic as I was constantly around these spaces in New York and I already had some set of observations from memory and conditioning.
